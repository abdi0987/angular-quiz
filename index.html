Welcome To Angular-Quiz

Angular quiz is a fun quiz for your users on your sites fully built using html,css,and javascript.

File Structure

(Main Dir)

  • angular-quiz
    • angular-quiz.js
    • angular-quiz.css
    • quiz.tmpl.html
  • index.html
  • app.js
  • package.json

Setup

Before you do anything you need to install the dependencies. To do that run npm install in your terminal.

Angular Quiz requires bootstrap and angular so you need to include them in your index file.

No in your javascript file you need to include angular-quiz module and set it up. 

var app = angular('app',['angular-quiz'])

Run

In the run method you need to requires the $quizConfig service

app.run(function($quizConfig){

})

and then call the setup method giving it an object.

for example.

app.run(function($quizConfig){
    $quizConfig.setup({
        questions : [
            {
                text:"Who invented the internet",
                answers:["Al Gore!!!!","Tim Burners Lee","Larry Page","Mark Zuckerberg"],
                correct:1
            },
            {
                text:"Who invented JavaScript",
                answers:["Brendan Eich","Whats Javascript","Again Larry Page"],
                correct:0
            },
            {
                text:"What does HTML stand for",
                answers:["HyperText Markup Language,","Hipster Target Man Language","IDK","HYPE!!"],
                correct:0
            },
            {
                text:"What company created angularjs",
                answers:['FaceBook',"Google",'Microsoft','Twitter','Apple','Dropbox','Spotify'],
                correct:1
            }
        ],
        prevAllowed:true
    })
})

The setup object should have a questions property which accepts an array of questions object. the prevAllow is optional its defaulted to true basically mean the user can go back to previous questions that they answered.

app.run(function($quizConfig){
    $quizConfig.setup({
        questions:[
                        {
                            text:"Who invented the internet",
                            answers:["Al Gore!!!!","Tim Burners Lee","Larry Page","Mark Zuckerberg"],
                            correct:1
                        },
                  ],
                  prevAllowed:false
        }
})

And thats it angular-quiz should appear.
